# Project : Number-Selector
This is a project in which the model selects a random number from 1-50 and the user has to predict which number the model has chosen. This is one of my beginner projects which I built to understand the concepts of random generations and improve my python logic.

Features : 
1. Interactive Number prediciton.
2. Simple and clean UI.
3. Fast execution.

How to Run :
1. Input the number you predict the model has chosen.
2. Based on how close or far your prediction was, the model will print a message to indicate your closeness.

Requirements :
Python 3.x
Jupyter Notebook

AI use :
Used for improving model idea a bit.
